This Project shows us the weather of any city present in the globe, just enter the name of your/any city from around the world to get the exact weather condtions.

I have used HTML for basic structure, CSS for some text, image formatting, and made this attractive using CSS and its elements. Also used Javascript for making my website behave or work as I wanted, and linked the open weather API which tells us the weather.
